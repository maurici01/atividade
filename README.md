# Atividade

**`Requisitos:`**
<p> Spring WEB, Spring JPA, H2, THYMELEAF

**`Instruções:`**

<p>Você deve criar um projeto Spring Boot que admita as seguintes funcionalidades:
<p>Exibir uma página html inicial em localhost:8080/projeto
<p>Exibir uma segunda página em localhost:8080/projeto/home
<p>No endpoint /projeto/getAll (tipo GET) exibir um JSON contendo todos os usuários. Considere que seu sistema contém 4 usuários. Cada usuário tem id, nome, email e telefone.
<p>Você não precisa implementar persistência em banco de dados. Se quiser, contudo, use o banco de dados H2.
